# 💗 Anniversary Getaway Page

A small interactive webpage created to share the details for an anniversary weekend getaway.

The page provides a relaxed overview of the trip so everything is easy to find in one place.

## Features

- Countdown to the getaway
- Weekend weather forecast for Liphook
- Airbnb location highlights
- Accordion sections for:
  - Location
  - Itinerary
  - What to wear
  - What to bring
- Simple responsive design
- Lightweight HTML, CSS and JavaScript

## Tech

Built using:

- HTML
- CSS
- Vanilla JavaScript
- Open-Meteo API for weather

No frameworks or build tools required.
